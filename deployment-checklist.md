# GitHub Pages 部署检查清单

## 文件结构检查
- [ ] 确保所有文件都已推送到GitHub仓库
- [ ] 检查文件名大小写是否正确（GitHub Pages区分大小写）
- [ ] 确保style.css文件位于正确位置

## 路径检查
- [ ] HTML中CSS链接路径是否正确：`<link rel="stylesheet" href="style.css">`
- [ ] CSS中引用的图片、字体等资源路径是否正确
- [ ] JavaScript文件路径是否正确

## 常见问题排查
- [ ] 在浏览器开发者工具中检查是否有404错误
- [ ] 检查控制台是否有CSS解析错误
- [ ] 确认GitHub Pages是否已正确启用（在仓库Settings中）

## 响应式设计测试
- [ ] 在不同设备尺寸下测试页面显示效果
- [ ] 检查媒体查询是否正常工作
- [ ] 确认触摸设备上的交互效果

## 浏览器兼容性
- [ ] 在不同浏览器中测试页面显示效果
- [ ] 检查是否使用了不兼容的CSS属性

## 性能优化
- [ ] 压缩CSS和JavaScript文件
- [ ] 优化图片大小和格式
- [ ] 减少HTTP请求数量
